Kebijakan Privasi

Data Lokal
Semua data Anda tersimpan sepenuhnya di perangkat Anda sendiri melalui browser localStorage. Kami tidak mengumpulkan, mengirim, atau menyimpan data pribadi Anda di server mana pun.


Keamanan Akun
Informasi akun (email dan password) disimpan secara lokal dan dienkripsi sederhana. Pastikan untuk melakukan ekspor data secara berkala sebagai cadangan.


Pembaruan
Kebijakan privasi dapat berubah sewaktu-waktu. Pengguna akan diberitahu melalui pembaruan aplikasi.
